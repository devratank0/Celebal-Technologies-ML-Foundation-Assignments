# 📝 Text Generation using RNN, LSTM & GRU

This repository contains my **Week 5 Machine Learning Foundation assignment** completed as part of the **Celebal Technologies Internship Program**.

## 📌 Project Overview

This project focuses on **Natural Language Processing (NLP)** and **text generation** using Recurrent Neural Networks. The project compares the performance of **Simple RNN**, **Long Short-Term Memory (LSTM)**, and **Gated Recurrent Unit (GRU)** models for next-word prediction and text generation. It includes text preprocessing, sequence generation, model training, performance comparison, and text generation using deep learning techniques.

## 🎯 Objectives

* Load and preprocess a text corpus
* Perform tokenization and vocabulary creation
* Generate input sequences using the sliding window (n-gram) approach
* Build and train a Simple RNN model
* Build and train an LSTM model
* Build and train a GRU model
* Compare model performance using training loss
* Generate text using all three trained models
* Analyze and compare the effectiveness of different recurrent neural network architectures

## 📈 Results

### 🧠 Simple RNN

* Successfully learned basic sequential text patterns and generated context-based text.

### 🔒 Long Short-Term Memory (LSTM)

* Achieved improved sequence learning by capturing long-term dependencies and generating more meaningful text.

### ⚡ Gated Recurrent Unit (GRU)

* Delivered competitive performance with a simpler architecture and efficient training compared to LSTM.

### 📊 Model Comparison

* Training loss comparison demonstrated the learning behavior of all three recurrent neural network architectures.
* Generated text samples highlighted the strengths of LSTM and GRU for sequence modeling tasks.
