# 🖼️ CIFAR-10 Image Classification using ANN & CNN

This repository contains my **Week 4 Machine Learning Foundation assignment** completed as part of the **Celebal Technologies Internship Program**.

## 📌 Project Overview

This project focuses on image classification using the **CIFAR-10** dataset and compares the performance of **Artificial Neural Networks (ANN)** and **Convolutional Neural Networks (CNN)**. The analysis includes image preprocessing, model development, performance evaluation, data augmentation, and architectural improvements to achieve better classification accuracy.

## 🎯 Objectives

* Load and explore the CIFAR-10 dataset
* Perform image preprocessing and normalization
* Build and evaluate an Artificial Neural Network (ANN)
* Build and evaluate a Convolutional Neural Network (CNN)
* Improve CNN architecture using deeper convolutional layers
* Apply Data Augmentation techniques
* Implement EarlyStopping to reduce overfitting
* Compare model performance using validation and test accuracy
* Generate a Classification Report for model evaluation

## 🔄 Workflow Followed

1. Import required libraries
2. Load the CIFAR-10 dataset
3. Explore the dataset and visualize sample images
4. Normalize image pixel values
5. Build and train an Artificial Neural Network (ANN)
6. Build and train a Convolutional Neural Network (CNN)
7. Improve CNN architecture using 32 → 64 → 128 convolutional filters
8. Apply EarlyStopping to prevent overfitting
9. Train a Data-Augmented CNN
10. Compare validation accuracy across all models
11. Compare final test accuracy using a performance table
12. Generate a Classification Report
13. Present final insights and conclusions

## 📈 Results

### 🤖 Artificial Neural Network (ANN)

* Successfully classified CIFAR-10 images with a strong baseline performance.

### 🧠 Convolutional Neural Network (CNN)

* Achieved significantly higher classification accuracy than the ANN by learning spatial image features.

### 🚀 Improved CNN

* Performance improved using deeper convolutional layers, Batch Normalization, Dropout, and EarlyStopping.

### 🎨 Data-Augmented CNN

* Data augmentation enhanced model generalization and reduced overfitting, resulting in improved classification performance.
