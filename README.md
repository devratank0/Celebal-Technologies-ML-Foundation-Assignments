# 📚 Weekly Assignments

| Week | Project | Topics Covered |
|------|---------|----------------|
| **Week 1** | Python Fundamentals | Python Programming, NumPy, Pandas, Data Analysis Fundamentals |
| **Week 2** | Tesla Deliveries Analysis & Forecasting | Exploratory Data Analysis (EDA), Feature Engineering, Regression Models, Time Series Analysis |

| **Week 4** | CIFAR-10 Image Classification | Artificial Neural Networks (ANN), Convolutional Neural Networks (CNN), Image Classification, Computer Vision |
| **Week 5** | Text Generation using RNN, LSTM & GRU | Natural Language Processing (NLP), Sequence Modeling, Recurrent Neural Networks, Deep Learning |
| **Week 6** | Denoising Autoencoder using MNIST | Autoencoders, Image Denoising, Encoder-Decoder Architecture, Convolutional Neural Networks (CNN), Computer Vision |
