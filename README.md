# 🚀 Celebal Technologies – Machine Learning Foundation Assignments

This repository contains all weekly assignments completed during the **Machine Learning Foundation Program** as part of the **Data Science Internship at Celebal Technologies**.

The assignments cover fundamental and advanced concepts of **Machine Learning**, **Deep Learning**, **Computer Vision**, and **Natural Language Processing (NLP)** using Python and TensorFlow/Keras.
